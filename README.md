**Cisco Packet Tracer Exam Monitoring**
The network infrastructure is designed in Cisco Packet Tracer with VLANs, Inter-VLAN routing, OSPF, NAT/PAT, GRE VPN tunnel connecting a remote branch, ACLs protecting the exam server, DHCP/DNS services, STP security, and VoIP with QoS prioritization

**Academic Context:** This project was developed as part of the "Database Managements System + computer networks" at Innovation University.
